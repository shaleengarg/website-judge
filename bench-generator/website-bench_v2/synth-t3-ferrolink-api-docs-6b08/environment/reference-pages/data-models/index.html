FerroLink / Data Models

Data Models

Canonical JSON schemas exchanged by the FerroLink API. All payloads are UTF-8 encoded; timestamps follow RFC 3339 with UTC offsets. Nested objects are documented inline with indentation matching the wire format.

StreamObject DeviceRecord AlertPayload TelemetryPoint AuthToken WebhookEvent
Fields marked * are required. Type annotations follow OpenAPI 3.1 (e.g. string<uuid>, integer<int64>).

StreamObject /v3/streams

Represents a single ingestion stream bound to one or more devices. Streams are the primary unit of throughput billing and retention policy.

FieldTypeDescription
id* string<uuid> Server-assigned identifier. Immutable after creation.
name* string Human-readable label, 3–64 chars. Must match ^[a-z0-9-]+$.
protocol* enum One of mqtt, amqp, http2, opcua.
retention object (optional) Retention policy. Defaults to 30-day rolling window if omitted.
retention.days integer Days to retain raw telemetry. Range 1–365.
retention.tier enum hot, warm, or cold.
created_at* string<datetime> RFC 3339 UTC timestamp of stream creation.
tags array<string> Up to 16 free-form labels for routing and dashboard grouping.

DeviceRecord /v3/devices

Describes a registered physical asset (PLC, gateway, sensor) authorized to publish into one or more streams.

FieldTypeDescription
device_id* string Manufacturer-assigned hardware serial. Globally unique.
model* string e.g. FL-Gateway-220, FL-Sensor-Mini.
firmware object Currently installed firmware metadata.
firmware.version string<semver> Reported via the boot handshake.
firmware.signed boolean Whether the image passed FerroLink secure-boot verification.
location object Optional physical placement metadata.
location.site string Plant or facility identifier.
location.coords array<number> Tuple [lat, lon] in decimal degrees.
status* enum online, offline, degraded, quarantined.

AlertPayload /v3/alerts

Emitted whenever a rule defined on a stream crosses its threshold. Delivered via webhook, polling endpoint, or fan-out queue.

FieldTypeDescription
alert_id* string<uuid> Idempotency key — safe to dedupe on this value.
severity* enum info, warning, critical, fatal.
rule_ref* string The rule slug that produced this alert.
context object Snapshot of metric values at trigger time.
context.metric string Metric name, e.g. temp.bearing.c.
context.value number Observed value that crossed the threshold.
context.threshold number Configured trigger threshold.
triggered_at* string<datetime> When the rule evaluator fired.

TelemetryPoint ingest

Atomic measurement record. Streams accept batched arrays of TelemetryPoint objects up to 5,000 entries per request.

FieldTypeDescription
t* integer<int64> Unix millisecond epoch. Server rejects values older than 7 days.
metric* string Dotted metric path. Max 128 chars.
value* number | boolean | string Sample value. Strings are truncated at 256 bytes.
unit string UCUM-compatible unit symbol (e.g. Cel, kPa, rpm).
quality integer OPC-UA-style quality code, 0–255. Defaults to 192 (good).
tags object<string,string> Up to 8 key/value labels for dimension fan-out.

AuthToken /v3/auth

Returned by the credential exchange endpoint. Bearer-formatted JWT with embedded scope claims.

FieldTypeDescription
access_token* string<jwt> Send as Authorization: Bearer <token>.
token_type* string Always Bearer in v3.
expires_in* integer Seconds until expiry. Refresh at 80% of lifetime.
scope array<string> Granted scopes, e.g. streams:read, devices:write.
refresh_token string Single-use; rotates on every exchange.

WebhookEvent delivery

Envelope wrapping any push notification sent to a subscriber endpoint. The data field holds the typed inner payload (e.g. an AlertPayload or DeviceRecord).

FieldTypeDescription
event_id* string<uuid> Stable identifier across retries.
type* string Dotted type, e.g. alert.triggered, device.online.
api_version* string Wire-format version, currently 2024-08-01.
data* object Inner payload. Schema determined by type.
delivery object Delivery metadata for debugging redelivery.
delivery.attempt integer 1-based attempt counter. Max 8 before dead-letter.
delivery.signature string HMAC-SHA256 of body; verify with subscriber secret.
