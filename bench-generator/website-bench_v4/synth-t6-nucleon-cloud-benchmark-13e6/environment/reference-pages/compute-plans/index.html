Burst window

Burst clock sustainable for ≤ 120s before throttling on Nano/Micro; unlimited on Pro & Enterprise dedicated cores.

DDR4 vs DDR5

DDR5 tiers achieve ~38% higher memory bandwidth on STREAM Triad benchmarks vs DDR4 generation hardware.

Hypervisor isolation

Xen-backed tiers expose CPU pinning and NUMA topology via metadata API; KVM tiers use noisy-neighbor mitigation only.

Billing granularity

All compute costs metered to the second with a 60-second minimum charge; sustained-use discount kicks in at 720h/mo.

Spot availability
