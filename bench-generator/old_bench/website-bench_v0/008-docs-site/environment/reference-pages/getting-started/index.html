Quickstart

Go from zero to a deployed application in five steps. This guide assumes you have a terminal and a project you'd like to deploy.

  1. Install the CLI

    Install orbit using the official install script. Works on macOS and Linux.

    curl -fsSL https://get.orbit.dev | sh

  2. Authenticate

    Log in to your orbit account. A browser window will open for authentication.

    orbit auth login
# Opening browser… authenticated as alex@example.com

  3. Initialize a project

    Run init inside your repository to generate orbit.yml.

    cd ./my-app
orbit init --template node

  4. Configure environment

    Add the secrets and variables your app needs. Values are encrypted at rest.

    orbit env set DATABASE_URL="postgres://…"
orbit env set NODE_ENV=production

  5. Deploy

    Ship to production. Subsequent deploys are incremental and typically finish in seconds.

    orbit deploy --env production
✓ Build completed in 12.4s
✓ Live at https://my-app.orbit.app
Tip: add orbit deploy to your CI to deploy automatically on every push to main.
