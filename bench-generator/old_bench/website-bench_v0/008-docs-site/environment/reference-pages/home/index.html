orbit

A fast, friendly CLI for shipping applications to any cloud. Zero config to start, infinitely configurable when you need it.

Install
# macOS / Linux
curl -fsSL https://get.orbit.dev | sh

# or with Homebrew
brew install orbit

# verify
orbit --version
orbit 1.2.0 (darwin/arm64)
Deploy your first project
$ orbit init my-app
$ cd my-app
$ orbit deploy --env production
⚡

Instant deploys

Push code and ship in under 30 seconds with incremental builds and edge caching.

⚙

Declarative config

Single orbit.yml describes services, secrets, and environments. Versioned with your code.

🛰

Multi-cloud

Deploy to AWS, GCP, Fly, or your own Kubernetes cluster. One command, same interface.
