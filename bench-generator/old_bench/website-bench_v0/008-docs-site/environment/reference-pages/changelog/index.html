Changelog

Notable changes to orbit. We follow semantic versioning — breaking changes only occur in major releases.

v1.2

LatestReleased March 14, 2024

Faster cold deploys, a redesigned plugin API, and broader Windows support.

Added
  • Native Windows binary (orbit.exe) with full feature parity.
  • New orbit doctor command to diagnose common setup issues.
  • Plugin API v2 with lifecycle hooks for prebuild and postdeploy.
Fixed
  • Resolved race condition causing duplicated log lines during high-throughput streams.
  • Fixed env var injection on Alpine-based images.

v1.1

StableReleased January 8, 2024

Quality-of-life improvements across deploys and the dashboard.

Changes
  • Incremental builds enabled by default — typical deploys are 3× faster.
  • Rollbacks now retain previous build artifacts for 30 days.
  • Improved error messages when orbit.yml is malformed.
  • Fixed crash when project name contained Unicode characters.

v1.0

StableReleased October 2, 2023

First stable release. Production-ready API with long-term support guarantees.

Highlights
  • Stable orbit.yml schema with versioned migrations.
  • Multi-region deploys to AWS, GCP, and Fly.
  • Encrypted secrets store with audit logging.
  • Public REST API and TypeScript SDK.
  • Documentation portal at docs.orbit.dev.
