Independent Benchmark Data — Q3 2024

Vaultline contracts Northridge Performance Labs to run quarterly, region-isolated workloads against each storage tier. The figures below cover sustained sequential throughput, small-block random IOPS, and edge-of-network time-to-first-byte latency across five global test regions. Top-scoring cells on each row are highlighted.

1. Sequential Read / Write Throughput (MB/s)

table_01 · units: MB/s · higher_is_better

Methodology. Each tier provisioned a single 4 TB volume on a fresh tenant; workloads issued 1 MiB sequential reads followed by writes using fio with iodepth=32, numjobs=8, and direct=1. Reported values are the median of three consecutive 10-minute runs, sampled at one-second granularity. Caches were dropped between runs and cross-region replication was disabled to isolate raw throughput.

2. Random 4K IOPS — Mixed 70/30 Read/Write

table_02 · units: IOPS · higher_is_better

Methodology. Small-block performance was measured with a 70% read / 30% write mix at 4 KiB block size, iodepth=64, numjobs=16, against a pre-conditioned 1 TiB volume to defeat sparse-allocation optimizations. Values represent steady-state IOPS sustained over the final 30 minutes of a 90-minute soak. Burst credits were exhausted prior to measurement on tiers where applicable.

3. Time-to-First-Byte Latency (ms)

table_03 · units: ms · lower_is_better

Methodology. TTFB was measured against the public HTTPS object endpoint for each tier using wrk from a separate measurement fleet co-located in the same metropolitan area as each region. Requests fetched a 64 KiB object, TLS handshake included. The table reports the p95 latency across 50,000 requests per cell; for this metric, lower values win, and the lowest p95 per region is highlighted.
