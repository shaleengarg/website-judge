Configuration

The ferroflux.yaml file defines pipeline behaviour, sensor hardware bindings, stream encoding, calibration profiles, and telemetry sinks. Values not present in the file fall back to the documented defaults below.

Example: minimal ferroflux.yaml

pipeline:
  name: "flux-edge-01"
  bufferSize: 4096
  sampleRateHz: 2000

sensor:
  interface: "spi0"
  model: "FX-3A"
  axes: ["x", "y", "z"]

stream:
  encoding: "cbor"
  endpoint: "wss://ingest.local/v1"
  # compression is optional
  compression: "zstd"
Resolution order: CLI flags → environment variables (FERROFLUX_*) → ferroflux.yaml → built-in defaults. Environment overrides always win over the YAML file.

Environment variable overrides

Every YAML key may be overridden at runtime by an environment variable using the FERROFLUX_ prefix and SCREAMING_SNAKE_CASE conversion of the dot-path. Variables are parsed using the same type rules as the YAML schema.

Environment variableMaps toNotes
FERROFLUX_PIPELINE_NAMEpipeline.nameString, no length limit.
FERROFLUX_PIPELINE_BUFFER_SIZEpipeline.bufferSizeInteger, must be a power of two.
FERROFLUX_PIPELINE_SAMPLE_RATE_HZpipeline.sampleRateHzInteger in Hz.
FERROFLUX_SENSOR_INTERFACEsensor.interfaceOverrides the transport at boot.
FERROFLUX_SENSOR_SERIALsensor.serialUse to pin a specific device.
FERROFLUX_STREAM_ENDPOINTstream.endpointFull URL including scheme.
FERROFLUX_STREAM_ENCODINGstream.encodingEnum value, lowercase.
FERROFLUX_STREAM_TLS_VERIFYstream.tls.verifySet to 0 to disable TLS verification.
FERROFLUX_CALIBRATION_PROFILEcalibration.profileSwitches profile without editing YAML.
FERROFLUX_CALIBRATION_AUTO_ZEROcalibration.autoZeroBoolean (0/1).
FERROFLUX_TELEMETRY_EXPORTERtelemetry.exporterUseful for swapping exporters per environment.
FERROFLUX_TELEMETRY_BIND_ADDRtelemetry.bindAddrFormat host:port.
FERROFLUX_TELEMETRY_LOG_LEVELtelemetry.logLevelOverrides the YAML value globally.
FERROFLUX_CONFIG_FILE— (loader)Path to an alternate YAML config file.
Validation: Run ferroflux config check ./ferroflux.yaml to lint the schema and resolve environment overrides without starting the pipeline. The command exits non-zero on any unknown key.
