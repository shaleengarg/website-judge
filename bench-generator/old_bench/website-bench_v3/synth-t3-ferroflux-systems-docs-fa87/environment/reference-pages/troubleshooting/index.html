FF-001 Sensor handshake times out after 5s on USB-HID init. udev rule missing for vendor ID 0x2f1c. Install 99-ferroflux.rules from SDK and reload udev. Resolved
FF-002 WebSocket disconnects every 60 seconds. Reverse-proxy idle timeout below keepalive interval. Set proxy_read_timeout 120s or enable ping frames. Resolved
FF-003 401 returned immediately after token refresh. Clock skew > 30s between client and auth server. Enable NTP sync; SDK retries once on skew detection. Resolved
FF-004 Streaming throughput drops below 200 Hz under load. Default pipeline.bufferSize too small for batch mode. Increase to 4096; tune flushInterval to 25ms. Investigating
FF-005 YAML loader rejects valid anchors and aliases. Strict-mode parser disallows merge keys by default. Set config.strict: false or flatten anchors. Resolved
FF-006 Calibration drift > 0.4% after firmware 3.2 upgrade. Vendor changed ADC gain step in 3.2 silently. Run ferroflux calibrate --reset-bias after upgrade. Investigating
FF-007 SPI bus reports EIO intermittently. Shared bus contention with display controller. Dedicate CS line; disable display HW-CS pin in DT overlay. Resolved
FF-008 API keys generated before v1.0.0 rejected. Legacy HMAC-SHA1 signatures deprecated. Rotate via POST /v1/auth/keys; legacy support removed. Wontfix
FF-009 Stream messages arrive out-of-order at high QPS. UDP fallback transport enabled in degraded mode. Force TCP via transport: tcp in client config. Resolved
FF-010 Env var overrides ignored on Windows hosts. Loader normalizes keys to lowercase incorrectly. Use FERROFLUX_* exactly; patched in v1.4.1. Resolved
FF-011 Sensor reports NaN readings near rare-earth magnets. Hall element saturated past ±80 mT range. Operate within rated range; enable saturation flag. Investigating
FF-012 CLI hangs on ferroflux stream --tail. TTY width detection blocks on non-interactive shells. Pass --no-tty or set COLUMNS=120. Resolved

Still stuck?

If your issue is not listed, generate a diagnostic bundle with ferroflux doctor --bundle and attach it to a support ticket. Bundles include sensor telemetry, last 200 log lines, and a redacted copy of your ferroflux.yaml.
