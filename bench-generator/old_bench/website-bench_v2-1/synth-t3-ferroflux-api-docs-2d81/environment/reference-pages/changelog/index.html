Changelog

// Version history for the FerroFlux real-time magnetic-field sensor platform.
v2.4.1CURRENT 2025-03-14
  • AddedPer-region failover header X-FF-Region-Fallback on /v2/sensors/poll.
  • ChangedIncreased free-tier rate limit window from 60s to 75s rolling.
  • AddedFieldVector.uncertainty field exposing measurement variance in µT².
  • ChangedWebSocket reconnect backoff now jittered (250ms → 8s).
v2.4.0 2025-02-02
  • AddedNew /v2/streams/replay endpoint for cold-storage playback up to 30 days.
  • AddedCalibrationProfile.driftModel with linear and Kalman options.
  • ChangedDefault streaming payload encoding from JSON to application/msgpack.
  • Deprecated/v1/sensors/snapshot — use /v2/sensors/poll instead.
// Example: opt-in to msgpack streams
POST /v2/streams/subscribe
Accept: application/msgpack
Authorization: Bearer ff_sk_live_•••
v2.3.3 2024-12-19
  • ChangedWebhook retries now use exponential backoff capped at 6 attempts over 24h.
  • AddedHMAC signature header X-FerroFlux-Signature-256 on all outbound webhooks.
  • RemovedLegacy MD5 signing scheme from webhook dispatcher.
v2.3.0 2024-10-30
  • AddedRegional endpoints for AP-South (Mumbai, Singapore).
  • AddedSensorReading.regionOrigin echoes the resolving datacenter.
  • ChangedAuth token TTL extended from 1h to 4h for service accounts.
  • DeprecatedapiKey query parameter — use Authorization header.
v2.2.0 2024-09-04
  • AddedBulk endpoint POST /v2/sensors/poll/batch (up to 256 sensor IDs).
  • AddedTypeScript SDK @ferroflux/sdk@2.2 with full type coverage.
  • ChangedError envelopes now include traceId for support correlation.
v2.1.0 2024-07-12
  • AddedReal-time WebSocket streams at wss://stream.ferroflux.io/v2.
  • AddedCalibration profiles persisted server-side per sensor.
  • RemovedLong-polling fallback endpoint /v2/sensors/longpoll.
v2.0.0 2024-05-01
  • ChangedFull rewrite of the public API surface — see migration guide.
  • AddedResource-oriented endpoints replacing RPC-style methods.
  • AddedOpenAPI 3.1 specification published at /v2/openapi.json.
  • RemovedXML response format. JSON only going forward.
