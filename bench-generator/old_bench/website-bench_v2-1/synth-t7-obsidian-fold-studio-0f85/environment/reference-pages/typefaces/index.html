04 — House Typefaces

Three Pairings.
One Tension Between Voice & Silence.

Every identity we make leans on one of three carefully tuned type systems. Each pairing balances a structural display face against a softer counterweight — built to hold space, set tone, and survive at scale.

PAIRING 01 / EDITORIAL

Anchor & Float

Display Serif
+ Geometric Sans

Display — Anchor

  • 96ptAa
  • 64ptAa Bb
  • 48ptAnchor
  • 32ptQuiet authority.
  • 18ptA weighted display serif.
  • 12ptFLOAT — geometric sans companion
  • 8ptCAPTIONS, METADATA, FINE PRINT SET IN FLOAT REGULAR
A · Tension
Anchor's heavy brackets read as architectural; Float's open counters release the page.
B · Use
Editorial covers, magazine mastheads, long-form annual reports.
C · Avoid
UI surfaces under 14pt — terminals are too narrow at small sizes.
PAIRING 02 / TECHNICAL

Blade & Reed

Condensed Grotesk
+ Humanist Mono

Display — Blade

  • 96ptBb
  • 64ptBb Rr
  • 48ptBLADE
  • 32ptSharp, narrow, decisive.
  • 18ptA condensed grotesk.
  • 12ptREED — humanist mono companion
  • 8ptspec sheets · indices · code listings
A · Tension
Blade's vertical compression sets against Reed's slightly humanist mono rhythm.
B · Use
Aerospace, architectural, charter and transit identities.
C · Avoid
Soft hospitality brands — the cap height reads too clinical.
PAIRING 03 / CEREMONIAL

Monolith & Script

Heavy Slab
+ Copperplate Italic

Display — Monolith

  • 96ptMm
  • 64ptMm Ss
  • 48ptMONOLITH
  • 32ptBuilt to outlast trends.
  • 18ptScript — a copperplate italic.
  • 12ptsignatures, dedications, certificates
  • 8ptCOLOPHON · IMPRINT · LEGAL NOTICES
A · Tension
Monolith's square slabs against Script's swashed copperplate descenders.
B · Use
Spirits, records, ateliers, ceramics — heritage-coded clients.
C · Avoid
Software products and dashboard contexts.
