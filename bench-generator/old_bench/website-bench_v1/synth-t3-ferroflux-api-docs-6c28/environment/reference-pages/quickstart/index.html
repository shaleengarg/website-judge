Quickstart

Get from zero to your first real-time sensor stream in under five minutes. This guide walks through installation, authentication, and publishing your first event to FerroFlux.

1

Install the FerroFlux CLI

The CLI is the fastest way to provision streams and inspect data locally. Install it with your package manager of choice — Node.js 18+ is required.

bash 
# Install via npm
npm install -g @ferroflux/cli

# Verify the installation
ferroflux --version
=> ferroflux/2.4.0 darwin-arm64 node-v20.11.0
2

Generate an API key

Authenticate the CLI against your workspace. This opens a browser session and writes a scoped key to ~/.ferroflux/credentials.

bash 
ferroflux auth login --workspace "acme-prod"
→ Opening https://app.ferroflux.io/auth/cli ...
✓ Authenticated as jordan@acme.io
✓ Wrote key fx_live_a1b2...e9f4 to credentials file
⚠
Keep your API key secret
Keys prefixed with fx_live_ grant full read/write access to your workspace. Never commit them to source control or embed them in client-side code. Use fx_test_ keys for development and rotate live keys every 90 days from the dashboard.
3

Create your first stream

Streams are typed channels of timestamped sensor readings. Create one for vibration data with a sampling hint of 200Hz.

bash 
curl -X POST https://api.ferroflux.io/v2/streams \
  -H "Authorization: Bearer $FERROFLUX_KEY" \
  -H "Content-Type: application/json" \
  -d '{ "name": "press-3-vibration", "unit": "g", "hz": 200 }'
4

Publish a sample reading

Push a single data point to confirm the pipeline is live. Production workloads should batch points and use the SDK's streaming client instead.

json 
{
  "stream_id": "strm_7QkPz2",
  "ts": "2025-01-14T09:24:11.482Z",
  "value": 0.0184,
  "tags": { "axis": "z", "line": "A" }
}
5

Tail the live feed

Subscribe to your stream and watch readings arrive in real time. The CLI uses Server-Sent Events under the hood, so it works behind most corporate proxies.

bash 
ferroflux streams tail strm_7QkPz2 --format compact
[09:24:11.482] 0.0184g  axis=z line=A
[09:24:11.487] 0.0192g  axis=z line=A
[09:24:11.492] 0.0179g  axis=z line=A
# Press Ctrl+C to disconnect
