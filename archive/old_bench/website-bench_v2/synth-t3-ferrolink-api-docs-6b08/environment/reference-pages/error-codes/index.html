Error Codes

A complete reference of all 47 HTTP and application-level error codes returned by the FerroLink ingestion gateway and control plane. Codes are grouped by category. Each short_name is the machine-readable identifier returned in the JSON body under the error.code field.

filter by code, name, or resolution…
All 4xx 429 5xx
Response shape
All errors return JSON with the structure: { "error": { "code": "...", "status": 0, "message": "...", "trace_id": "..." } }. Include the trace_id when contacting support to expedite diagnosis.
