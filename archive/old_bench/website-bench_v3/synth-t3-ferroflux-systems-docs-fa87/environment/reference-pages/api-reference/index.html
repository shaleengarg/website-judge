API Reference

Full REST and WebSocket endpoint reference for the FerroFlux SDK v1.4.2. Endpoints are grouped by resource. All requests require an Authorization: Bearer token unless noted.

Sensors

Discover, query, and manage connected magnetic sensor devices on USB-HID or SPI interfaces.

List sensors

GET /v1/sensors

Returns a paginated list of all sensors registered on the local pipeline daemon.

NameTypeRequiredDescription
limitintegernoPage size, max 200. Default 50.
cursorstringnoOpaque pagination token from a previous response.
interfaceenumnoFilter by usb-hid or spi.
cURL
curl -X GET \
  https://api.ferroflux.io/v1/sensors \
  -H "Authorization: Bearer $TOK"
200 OK
{
  "sensors": [
    {
      "id": "sx-0a3f",
      "interface": "usb-hid",
      "status": "online"
    }
  ],
  "cursor": "eyJwIjoyfQ=="
}

Get sensor

GET /v1/sensors/{sensorId}

Retrieve detailed metadata and live status for a single sensor.

NameTypeRequiredDescription
sensorIdstringyesSensor identifier, e.g. sx-0a3f.
expandstring[]noComma-separated. Use firmware,metrics.
cURL
curl https://api.ferroflux.io/v1/sensors/sx-0a3f \
  -H "Authorization: Bearer $TOK"
200 OK
{
  "id": "sx-0a3f",
  "model": "FF-HMC5983",
  "axes": 3,
  "sampleRateHz": 800,
  "status": "online"
}

Register sensor

POST /v1/sensors

Registers a new sensor device with the pipeline. Returns 409 if the hardware ID already exists.

NameTypeRequiredDescription
hardwareIdstringyesHex serial number from the device.
interfaceenumyesusb-hid or spi.
labelstringnoHuman-readable name, max 64 chars.
cURL
curl -X POST \
  https://api.ferroflux.io/v1/sensors \
  -H "Authorization: Bearer $TOK" \
  -d '{"hardwareId":"0xA31F",
       "interface":"usb-hid"}'
201 Created
{
  "id": "sx-0a3f",
  "created": "2024-09-12T14:02Z"
}

Delete sensor

DELETE /v1/sensors/{sensorId}

Removes a sensor and tears down any open streams. Idempotent — returns 204 even if not found.

NameTypeRequiredDescription
sensorIdstringyesSensor identifier.
forcebooleannoSkip graceful disconnect. Default false.
cURL
curl -X DELETE \
  https://api.ferroflux.io/v1/sensors/sx-0a3f \
  -H "Authorization: Bearer $TOK"
204 No Content
// empty body

Streams

Real-time WebSocket and REST endpoints for high-frequency magnetic flux readings.

Open stream

WS /v1/streams/{sensorId}/live

Bidirectional WebSocket. Server pushes flux frames at the configured sample rate; client may send control frames to adjust filters.

NameTypeRequiredDescription
sensorIdstringyesTarget sensor ID.
formatenumnojson (default) or cbor.
downsampleintegernoEmit every Nth sample. Min 1.
Handshake
wscat -c \
  wss://api.ferroflux.io/v1/streams/sx-0a3f/live \
  -H "Authorization: Bearer $TOK"
Frame
{
  "t": 1736290110.214,
  "x": 12.41,
  "y": -3.07,
  "z": 48.92
}

List streams

GET /v1/streams

Returns active stream sessions across all sensors with subscriber counts.

NameTypeRequiredDescription
sensorIdstringnoFilter to streams for one sensor.
stateenumnoactive, paused, closed.
cURL
curl https://api.ferroflux.io/v1/streams?state=active \
  -H "Authorization: Bearer $TOK"
200 OK
{
  "streams": [
    {
      "id": "st-91c2",
      "sensor": "sx-0a3f",
      "subscribers": 3
    }
  ]
}

Pause stream

POST /v1/streams/{streamId}/pause

Suspends frame emission while keeping the session and buffer alive. Resume with /resume.

NameTypeRequiredDescription
streamIdstringyesStream session identifier.
drainMsintegernoGrace window to flush buffered frames.
cURL
curl -X POST \
  https://api.ferroflux.io/v1/streams/st-91c2/pause \
  -H "Authorization: Bearer $TOK"
200 OK
{
  "id": "st-91c2",
  "state": "paused"
}

Close stream

DELETE /v1/streams/{streamId}

Terminates the stream session and disconnects all subscribers. Buffered samples are discarded.

NameTypeRequiredDescription
streamIdstringyesStream session identifier.
cURL
curl -X DELETE \
  https://api.ferroflux.io/v1/streams/st-91c2 \
  -H "Authorization: Bearer $TOK"
204 No Content
// empty body

Calibration

Submit, fetch, and apply calibration matrices to compensate for soft- and hard-iron offsets.

Get calibration

GET /v1/calibration/{sensorId}

Returns the current 3×3 soft-iron matrix and hard-iron offset vector for a sensor.

NameTypeRequiredDescription
sensorIdstringyesTarget sensor.
revisionintegernoFetch a historical calibration revision.
cURL
curl https://api.ferroflux.io/v1/calibration/sx-0a3f \
  -H "Authorization: Bearer $TOK"
200 OK
{
  "softIron": [[1.02,0,0],
                [0,0.98,0],
                [0,0,1.01]],
  "hardIron": [-2.1,0.4,11.7]
}

Submit calibration

PUT /v1/calibration/{sensorId}

Uploads a new calibration. The previous revision is retained for rollback for 30 days.

NameTypeRequiredDescription
sensorIdstringyesTarget sensor.
softIronfloat[3][3]yes3×3 correction matrix.
hardIronfloat[3]yesBias offset vector in µT.
notestringnoFree-form revision note.
cURL
curl -X PUT \
  https://api.ferroflux.io/v1/calibration/sx-0a3f \
  -H "Authorization: Bearer $TOK" \
  -d @cal.json
200 OK
{
  "revision": 7,
  "applied": "2024-09-12T14:18Z"
}

Start auto-calibration

POST /v1/calibration/{sensorId}/auto

Initiates a guided figure-8 auto-calibration routine. Progress is reported via the live stream.

NameTypeRequiredDescription
sensorIdstringyesTarget sensor.
durationSecintegernoCollection window, default 30.
minSamplesintegernoMinimum samples required to accept fit.
cURL
curl -X POST \
  https://api.ferroflux.io/v1/calibration/sx-0a3f/auto \
  -H "Authorization: Bearer $TOK"
202 Accepted
{
  "jobId": "cal-job-44a",
  "status": "collecting"
}

Reset calibration

DELETE /v1/calibration/{sensorId}

Reverts the sensor to factory-default calibration. Active streams will see corrected values change immediately.

NameTypeRequiredDescription
sensorIdstringyesTarget sensor.
keepHistorybooleannoPreserve prior revisions. Default true.
cURL
curl -X DELETE \
  https://api.ferroflux.io/v1/calibration/sx-0a3f \
  -H "Authorization: Bearer $TOK"
200 OK
{
  "reset": true,
  "revision": 0
}

Auth

Issue, rotate, and revoke bearer tokens. Tokens are scoped per-pipeline and expire after 24 hours by default.

Issue token

POST /v1/auth/token

Exchanges an API key for a short-lived bearer token. This is the only unauthenticated endpoint.

NameTypeRequiredDescription
apiKeystringyesKey issued via the FerroFlux console.
scopesstring[]noSubset of read, write, admin.
ttlSecintegernoLifetime in seconds, max 86400.
cURL
curl -X POST \
  https://api.ferroflux.io/v1/auth/token \
  -d '{"apiKey":"ff_live_..."}'
200 OK
{
  "token": "eyJhbGciOiJI...",
  "expiresIn": 86400,
  "scopes": ["read","write"]
}

Introspect token

GET /v1/auth/introspect

Returns the active token's scopes, subject, and expiry. Useful for SDK debug logs.

NameTypeRequiredDescription
AuthorizationheaderyesBearer token to introspect.
cURL
curl https://api.ferroflux.io/v1/auth/introspect \
  -H "Authorization: Bearer $TOK"
200 OK
{
  "active": true,
  "sub": "pipeline-prod-1",
  "exp": 1736376510
}

Revoke token

DELETE /v1/auth/token

Immediately invalidates the supplied bearer token. Open WebSocket sessions are closed within 2 seconds.

NameTypeRequiredDescription
AuthorizationheaderyesBearer token to revoke.
cascadebooleannoRevoke all sibling tokens with the same subject.
cURL
curl -X DELETE \
  https://api.ferroflux.io/v1/auth/token \
  -H "Authorization: Bearer $TOK"
204 No Content
// empty body
