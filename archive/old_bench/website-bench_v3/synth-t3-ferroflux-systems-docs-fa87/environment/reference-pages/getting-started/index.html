Getting Started

Welcome to FerroFlux Systems. This guide walks you through installing the SDK, initializing your first sensor pipeline, and streaming a live magnetic flux reading in under five minutes. By the end, you'll have a working ferroflux-sdk instance emitting calibrated tesla readings from a connected USB-HID or SPI sensor.

Minimum System Requirements

  • Node.js 18.0 or newer (LTS recommended)
  • Linux kernel 5.15+ with hidraw module loaded
  • USB-HID or SPI hardware interface
  • ~120 MB free RAM for the streaming buffer
1

Install via npm

FerroFlux ships as a single npm package with native bindings prebuilt for x64 and arm64 Linux targets. Install it into your project with your preferred package manager:

# install the SDK and its peer types
npm install ferroflux-sdk

# or with pnpm / yarn
pnpm add ferroflux-sdk
yarn add ferroflux-sdk

The post-install step probes /dev/hidraw* and verifies kernel compatibility. If detection fails, see Troubleshooting → Hardware.

2

Initialize a sensor pipeline

Create a ferroflux.yaml manifest at the root of your project. The minimum schema declares one sensor source and one output stream:

pipeline:
  name: "primary-flux"
  bufferSize: 4096
  sampleRateHz: 1000

sensors:
  - id: "mag-01"
    interface: "usb-hid"
    vendorId: "0x2341"
    calibrationProfile: "default-3axis"

streams:
  - name: "live"
    format: "json"
    transport: "websocket"
    port: 8787

Key fields

Field Type Description
pipeline.bufferSize integer Ring-buffer depth in samples. Power-of-two recommended.
pipeline.sampleRateHz integer Target acquisition rate; clamped to sensor capability.
sensors[].interface enum One of usb-hid, spi, mock.
streams[].transport enum websocket, tcp, or stdout.
3

Stream the first reading

The snippet below loads your manifest, attaches an event listener, and prints the first ten calibrated samples to stdout. Save it as index.js and run with node index.js.

import { FerroFlux } from "ferroflux-sdk";

const ff = await FerroFlux.fromConfig("./ferroflux.yaml");

await ff.connect();
await ff.calibrate({ profile: "default-3axis" });

let count = 0;
ff.streams.live.on("sample", (sample) => {
  console.log(`[${sample.t}] x=${sample.x}µT y=${sample.y}µT z=${sample.z}µT`);
  if (++count >= 10) ff.stop();
});

await ff.start();

A successful run will log ten lines of three-axis flux density readings in microtesla, sampled at the rate declared in your manifest.
