REST Endpoints

The FerroFlux REST API exposes resources for managing sensor streams and retrieving real-time telemetry. All endpoints are rooted at https://api.ferroflux.io/v2 and require a Bearer token.

List Streams

GET /streams

Returns a paginated collection of sensor streams accessible to the authenticated workspace.

Query Parameters

NameTypeDescription
limitintegerMax items per page (1–200). Defaults to 50.
cursorstringOpaque cursor returned by the previous response.
regionstringFilter streams by deployment region code.
Request
cURL# list streams in eu-west-1 curl -X GET "https://api.ferroflux.io/v2/streams?region=eu-west-1&limit=20" \ -H "Authorization: Bearer $FF_TOKEN"
Response · 200 OK
JSON{ "data": [ { "id": "strm_8x2qK", "name": "turbine-04", "region": "eu-west-1", "rate_hz": 240 } ], "next_cursor": "eyJvIjoyMH0=" }

Create Stream

POST /streams

Provisions a new sensor stream and returns its ingest credentials. The stream becomes active within ~3s.

Body Parameters

NameTypeDescription
namestringHuman-readable stream label, max 64 chars.
rate_hzintegerTarget sample rate. Supports 1–10000.
schemaobjectTyped map of field names to FerroFlux primitive types.
retention_daysintegerOptional. Days to retain raw samples. Default 30.
Request
JSONPOST /v2/streams { "name": "turbine-05", "rate_hz": 240, "schema": { "rpm": "float32", "temp": "float32" } }
Response · 201 Created
JSON{ "id": "strm_9p4tL", "name": "turbine-05", "ingest_url": "wss://ingest.ferroflux.io/strm_9p4tL", "ingest_key": "ik_live_a1b2c3...", "created_at": "2024-09-14T10:21:08Z" }

Read Stream Data

GET /streams/{id}/data

Fetches a time-bounded window of samples from a stream. Returns up to 100k points per request.

Path & Query Parameters

NameTypeDescription
idstringStream identifier (path).
fromtimestampISO-8601 lower bound, inclusive.
totimestampISO-8601 upper bound, exclusive.
downsamplestringAggregation interval, e.g. 1s, 500ms.
Request
cURLcurl -X GET "https://api.ferroflux.io/v2/streams/strm_8x2qK/data?from=2024-09-14T09:00Z&to=2024-09-14T09:05Z&downsample=1s" \ -H "Authorization: Bearer $FF_TOKEN"
Response · 200 OK
JSON{ "stream_id": "strm_8x2qK", "points": [ ["2024-09-14T09:00:00Z", 1820.4, 63.1], ["2024-09-14T09:00:01Z", 1821.0, 63.2] ], "columns": ["ts", "rpm", "temp"] }

Delete Stream

DELETE /streams/{id}

Permanently removes a stream and all of its retained samples. This action is irreversible.

Path Parameters

NameTypeDescription
idstringStream identifier to delete.
forcebooleanIf true, skips the 24h soft-delete grace window.
reasonstringOptional audit string, recorded in workspace logs.
Request
cURLcurl -X DELETE "https://api.ferroflux.io/v2/streams/strm_8x2qK?force=true" \ -H "Authorization: Bearer $FF_TOKEN"
Response · 204 No Content
HTTP# empty body HTTP/1.1 204 No Content x-ferroflux-request-id: req_7Hk2pQ x-ferroflux-deleted-at: 2024-09-14T10:42:17Z
