Data Types

Canonical object shapes exchanged with the FerroFlux real-time magnetic-field API.

The FerroFlux platform exchanges six core data objects across REST endpoints and streaming sessions. All objects are encoded as application/json and validated against the published JSON Schema below. Field types use TypeScript notation; optional fields are marked with ?.

SensorReading

A single point-in-time measurement from a calibrated magnetic-field sensor.
idstring
sensorIdstring
timestampISO8601
fieldFieldVector
magnitudenumber
unit"uT" | "mT"
temp?number

FieldVector

Three-axis magnetic flux density tuple in the sensor's local reference frame.
xnumber
ynumber
znumber
frame"local" | "world"
precision"f32" | "f64"

CalibrationProfile

Bias and scale corrections applied to raw readings prior to publishing.
profileIdstring
createdAtISO8601
biasFieldVector
scaleFieldVector
softIronnumber[9]
validboolean

StreamSession

Long-lived WebSocket subscription metadata for live sensor channels.
sessionIdstring
channelsstring[]
openedAtISO8601
rateHznumber
backpressure"drop" | "block"
token?string

WebhookPayload

Signed event envelope delivered to registered webhook endpoints.
eventstring
deliveryIdstring
sentAtISO8601
dataSensorReading
signaturestring
attemptnumber

ErrorEnvelope

Uniform error response returned for any non-2xx status code.
codestring
statusnumber
messagestring
requestIdstring
details?object
retryableboolean

Canonical SensorReading Schema

JSON Schema draft 2020-12. Use this for client-side validation or codegen.
// GET /v2/schemas/sensor-reading.json
{
  "$schema": "https://json-schema.org/draft/2020-12/schema",
  "$id": "https://api.ferroflux.io/v2/schemas/sensor-reading.json",
  "title": "SensorReading",
  "type": "object",
  "required": ["id", "sensorId", "timestamp", "field", "magnitude", "unit"],
  "properties": {
    "id":        { "type": "string", "format": "uuid" },
    "sensorId":  { "type": "string", "pattern": "^ff-[a-z0-9]{12}$" },
    "timestamp": { "type": "string", "format": "date-time" },
    "field": {
      "type": "object",
      "required": ["x", "y", "z"],
      "properties": {
        "x": { "type": "number" },
        "y": { "type": "number" },
        "z": { "type": "number" },
        "frame": { "enum": ["local", "world"], "default": "local" }
      }
    },
    "magnitude": { "type": "number", "minimum": 0 },
    "unit":      { "enum": ["uT", "mT"] },
    "temp":      { "type": "number", "minimum": -40, "maximum": 125 }
  },
  "additionalProperties": false
}
