PANEL E 05

Rail-Grade Glossary

  • CLASS I Operating revenue ≥ $943M. Trunk-line carriers; 7 active in North America.
  • CLASS II Regional rail, $42M–$943M. Focus of this report's structural audit.
  • CLASS III Short-line / terminal carriers. Often single-shipper dependent.
  • FRA T-1 / T-2 Track classes limiting freight to 10/25 mph; speed restriction as proxy for decay.
  • SD / FO Structurally Deficient / Functionally Obsolete — federal bridge ratings.
  • LOAD RATING Max axle load (tons) the structure can carry without programmed repair.
